# GitHub 徽章生成链接

## 复制以下 Markdown 代码到 README.md

### 基础徽章

```markdown
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.7+](https://img.shields.io/badge/python-3.7+-blue.svg)](https://www.python.org/downloads/)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](CONTRIBUTING.md)
[![GitHub stars](https://img.shields.io/github/stars/YOUR_USERNAME/dwg2kml.svg?style=social&label=Star)](https://github.com/YOUR_USERNAME/dwg2kml)
[![GitHub forks](https://img.shields.io/github/forks/YOUR_USERNAME/dwg2kml.svg?style=social&label=Fork)](https://github.com/YOUR_USERNAME/dwg2kml)
[![GitHub issues](https://img.shields.io/github/issues/YOUR_USERNAME/dwg2kml.svg)](https://github.com/YOUR_USERNAME/dwg2kml/issues)
```

### 替换 YOUR_USERNAME 为你的 GitHub 用户名

例如：`YOUR_USERNAME/dwg2kml` → `wnp1234/dwg2kml`
