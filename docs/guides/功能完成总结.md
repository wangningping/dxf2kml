## 🚀 快速开始

### 安装依赖
```bash
# Windows
install.bat

# 手动安装
pip install -r requirements.txt
```

### 命令行使用
```bash
# 基本转换
python dwg2kml.py input.dxf -o output.kml

# CGCS2000 转换（北京地区）
python dwg2kml.py input.dxf -o output.kml --cgcs2000-3deg 39

# 批量转换
python dwg2kml.py --batch ./dxf_folder -o ./kml_output
```

### GUI 使用
```bash
# Windows
启动转换器.bat

# 或直接运行
python dwg2kml_gui.py
```
