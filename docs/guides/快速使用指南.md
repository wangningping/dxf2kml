# DWG2KML 快速使用指南

## 一、安装依赖

双击运行 `install.bat` 或手动安装：

```bash
pip install ezdxf simplekml pyproj -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 二、启动程序

双击 `启动转换器.bat` 或运行：
```bash
python dwg2kml_gui.py
```

## 三、配置坐标系

### 📋 数据信息

1. **数据路径** - 点击"选择"按钮选择 DXF 文件
2. **坐标系来源** - 选择"自定义配置"（已知坐标系）或"坐标系未知，快速配准"

### 🌐 坐标系信息

#### 1. 投影类型

- **高斯克吕格 (Gauss-Kruger)** - 中国地区推荐
- **UTM** - 国际项目使用

#### 2. 椭球参数

从下拉列表选择：
- 国家 2000 (CGCS2000) - 中国法定坐标系
- WGS 84 - GPS 全球坐标系
- 北京 54 - 旧坐标系
- 西安 80 - 旧坐标系

#### 3. 坐标系

从下拉列表选择预设坐标系，格式为：
```
国家 2000 高斯三度分带 中央经线 XXX°E (坐标不含分带)
```

#### 4. 中央子午线（度分秒格式）

可直接输入或从坐标系自动填充：
- **度°** - 整数部分
- **分′** - 0-59
- **秒″** - 0.00-59.99

点击 **→** 按钮可从中央子午线计算带号并更新坐标系

#### 5. 投影参数

- **投影面高程** - 平均海拔高度（米），默认 0.00
- **假东** - 东方向偏移（米），默认 500000.00
- **假北** - 北方向偏移（米），默认 0.00

## 六、中央子午线计算

### 3 度带
- 带号 = 中央经线 / 3（四舍五入）
- 中央经线 = 带号 × 3

### 6 度带
- 带号 = (中央经线 + 3) / 6（取整）
- 中央经线 = 带号 × 6 - 3

## 七、转换步骤

1. **选择输入文件** - 点击"选择"按钮
2. **配置坐标系** - 根据项目位置设置中央子午线等参数
3. **设置输出文件** - 选择 KML 保存位置
4. **点击"开始转换"** - 等待转换完成
5. **用 Google Earth 打开** - 查看结果

## 八、常见问题

### Q: 如何知道中央子午线？
A: 
- 查看 CAD 图纸标题栏
- 或根据项目位置查上表
- 或使用度分秒输入后点击 → 自动计算

### Q: 坐标含分带号怎么办？
A: 
- 如果坐标 Y 值包含带号前缀（如 39500000）
- 需要先在 CAD 中去掉前缀，保留实际坐标（如 500000）
- 或选择"坐标含分带"的坐标系选项

### Q: 假东/假北是什么？
A: 
- 假东：为避免负坐标，东方向加的常数（中国通常用 500000 米）
- 假北：为避免负坐标，北方向加的常数（北半球通常为 0）

### Q: 投影面高程需要设置吗？
A: 
- 一般地区可设为 0
- 高海拔地区可设置当地平均海拔，提高精度

## 九、命令行模式

```bash
# 单个文件
python dwg2kml.py input.dxf -o output.kml --cgcs2000-3deg 39

# 批量转换
python dwg2kml.py --batch ./dxf_files --cgcs2000-3deg 39

# 查看帮助
python dwg2kml.py --help
```
