## 🎯 查看方式

### 方法 1: 使用菜单

2. 点击菜单 **"帮助"**
3. 选择 **"关于"**
4. 查看开发者信息

### 方法 2: 使用快捷键
1. 启动 GUI
2. 按 **F1** 打开使用说明
3. 关闭后再次点击 **"帮助" → "关于"**
